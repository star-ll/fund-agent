## 数据来源

本项目使用akshare(python)作为数据来源，通过python + akshare实现数据接口层

安装：

```
uv pip install --upgrade akshare
```

akshare api 文档： [akshare文档](https://akshare.akfamily.xyz/tutorial.html)

## 架构设计

### 目录

目录支持扩展和调整，调整后同步更新本文档

server/
src/
    utils/
    prompts/
    tools/
knowledge/

### 架构层

- UI层（终端）
- API层（http接口）
- 数据层（akshare）
- 工具层（网络、文件、数据等）
- Agent层
    - Planner（规划）
    - Executor（执行 tools）
    - Reasoner（推理）
    - Reporter（输出结构化结果）
