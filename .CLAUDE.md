## 数据来源

本项目使用akshare(python)作为数据来源，通过python + akshare实现数据接口层

安装：

```
uv pip install --upgrade akshare
```

akshare api 文档： [akshare文档](https://akshare.akfamily.xyz/tutorial.html)

## 架构设计

### 目录

目录支持扩展和调整，调整后同步更新本文档

server/

    utils/
    prompts/
    tools/

### 架构层

- UI层（终端）
- API层（http接口）
- 数据层（akshare）
- 工具层（网络、文件、数据等）
- Agent层
    - Planner（规划）
    - Executor（执行 tools）
    - Reasoner（推理）
    - Reporter（输出结构化结果）

# 开发规范

## Git 工作流

每次新功能必须从主分支切出新分支开发，完成后合并回主分支并推送：

```bash
git checkout master
git pull
git checkout -b <feature-branch>
# 开发...
git checkout master
git merge <feature-branch>
git push
```

分支命名建议：`feature/<简短描述>`，如 `feature/user-profile-enrichment`。
